# the-punch

## Overview
*The Punch* is a social media based off the fictional social media *The Slap* from Victorious. It's a new social platform that allows friends to update each other on their day in real time without the pressure of having a "perfect profile". 

Our app that sends out a single push notification each day, prompting users to share what they are feeling or doing in the moment. By limiting posting to once per day, the app creates a simple, low-pressure environment for authentic sharing. The primary clients are friend groups, families, clubs, and organizations that want to stay connected and engaged in each other’s daily lives. It offers an easy way to maintain closeness without the stress and comparison often tied to traditional social media platforms. 

## Framework
**Frontend:** Swift/SwiftUI,
**Backend:** Node.JS + Express, MySQL database

## Basic User WorkFlow
1. User gets a push notification at a random point in the day saying that it's time to post a *punch*
2. User creates their punch consisting of an emoji, a feeling, and a short caption
3. User taps 'post'
4. The post will appear on their friend's feeds. NOTE: Users can only see their feed if they have posted their punch for the day.
